\doxysubsection{Detailed Description}
A wrapper class for managing ONNX model inference using ONNX Runtime. 

This class initializes an ONNX Runtime session, manages input/output tensors, and provides an interface for running inference on input data. The implementation supports flexible tensor shapes and data types.
