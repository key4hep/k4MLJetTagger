\doxysubsection{Detailed Description}
Gaudi transformer that builds edm4hep\+::\+Particle\+IDCollection objects (!plural, one for each flavor) for each jet in edm4hep\+::\+Reconstructed\+Particle\+Collection. 

We retrieve a description of the jet constituents which serve as an input to a neural network. The network is loaded as an ONNX model. The inference is run on each jet. The output of the network is a vector of probabilities for each jet flavor. We create one Particle\+ID collection per flavor create, link it to the jet and set the likelihood and PDG number.
